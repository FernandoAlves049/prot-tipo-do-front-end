# Sistema de Gestão Inteligente: Compras, Impostos e Projetos
**Entrega Final**
Equipe: Fernando Alves de Souza, Felipe Montalvão, Victor Hugo
Data: 25/03/2026

---

## 1. Introdução e Problematização

- **O Problema:** Empresas lutam com múltiplos sistemas desconexos para compras, carga tributária e projetos.
- **O Impacto:** Falta de visibilidade unificada de tarefas, status de compras e custos do projeto.
- **A Solução:** Uma aplicação Web Integrada (SPA) desenvolvida com React e Vite, centralizando o controle e as informações.

---

## 2. A Solução e Módulos Centrais

Uma interface moderna e altamente responsiva dividida em 4 pilares:

1. **Dashboard Analítico**: Visão geral de métricas.
2. **Gestão de Compras**: Cadastro de produtos e listas.
3. **Inteligência Tributária**: Consulta e cálculo de impostos.
4. **Gerenciamento de Projetos**: Tarefas, equipe e financeiro.

---

## 3. Tecnologias e Recursos (Tópico 6)

- **Frontend Core**: React (TypeScript), Vite.
- **Estilização e UI**: Tailwind CSS.
- **Gráficos**: ApexCharts para visualização de dados.
- **Roteamento & Estado**: React Router v6, Context API.
- **Recursos da Equipe**: Computador 16GB RAM, VS Code, Figma, Node.js + NPM, Git.
